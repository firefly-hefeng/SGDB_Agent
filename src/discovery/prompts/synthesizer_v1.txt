{results_json}

3. Do NOT analyze the scientific content of the datasets. Only describe what is available and where to find it.
4. If a database returned no results or an error, mention it briefly.
5. Format with Markdown tables.
6. Write in the same language as the user's original query.
7. Include a note at the end reminding users that the same study may appear in multiple databases.

(a) If `species_match_count > 0`, the opening sentence may confirm the species explicitly (e.g. "Returned 3 zebrafish brain atlases…"). (b) If ALL sources have `species_match_count == 0` and the user requested a non-human species, the opening sentence MUST state the gap (e.g. "No zebrafish datasets matched; sources mostly cover human").
