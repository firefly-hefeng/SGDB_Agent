## ✅ 已实现功能

### 后端API (app.py)

| API端点 | 方法 | 功能 |
|---------|------|------|
| `/api/health` | GET | 健康检查 |
| `/api/schema` | GET | 获取数据库Schema |
| `/api/query` | POST | 执行自然语言查询 |
| `/api/query/history` | GET | 获取会话历史 |
| `/api/statistics/{field}` | GET | 字段统计分析 |
| `/api/download/preview` | POST | 下载预览 |
| `/api/download/tasks` | POST | 创建下载任务 |
| `/api/download/start` | POST | 开始下载 |
| `/api/download/script` | POST | 生成下载脚本 |
| `/api/download/tasks` | GET | 获取下载任务列表 |
| `/api/field/expand` | POST | 字段扩展 |

### WebSocket事件

| 事件 | 方向 | 功能 |
|------|------|------|
| `connect` | C→S | 连接建立 |
| `query_stream` | C→S | 流式查询 |
| `query_result` | S→C | 查询结果推送 |
| `download_progress` | S→C | 下载进度更新 |
| `download_complete` | S→C | 下载完成通知 |

## 🚀 快速开始

### 1. 安装依赖

```bash
cd web
pip install -r requirements.txt
```

### 2. 启动服务

```bash
# 方式1: 直接启动
python app.py

# 方式2: 使用启动脚本
python start_server.py --port 5000
```

### 3. 访问界面

打开浏览器: http://localhost:5000

## 🔧 配置选项

### 环境变量

```bash
FLASK_ENV=development        # 运行环境
FLASK_DEBUG=1               # 调试模式
FLASK_PORT=5000             # 服务端口
```

### 配置文件

Web服务读取 `config/config.yaml` 中的配置：

```yaml
# 下载配置
download:
  download_dir: "data/downloads"
  max_concurrent_downloads: 3
  
# AI配置
ai:
  primary:
    provider: "kimi"
    api_key: "your-api-key"
```
