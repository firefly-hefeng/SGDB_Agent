# SCDB-Agent Web界面 - 5分钟快速开始

## 🚀 最快启动方式

### 第1步：安装依赖（1分钟）

```bash
cd web
pip install flask flask-cors flask-socketio python-socketio eventlet
```

### 第2步：启动服务（1分钟）

```bash
python app.py
```

### 第3步：打开浏览器（立刻）

访问：http://localhost:5000

---

## 🎯 第一个查询

1. 在搜索框输入：`查找肺癌相关的单细胞数据`
2. 点击"搜索"按钮
3. 查看返回的结果
4. 勾选感兴趣的数据
5. 点击"下载数据"

---

## 🌐 分享给他人

### 方法1: 同一WiFi下分享

```bash
# 启动时加上 --host 0.0.0.0
python start_server.py --host 0.0.0.0 --port 5000

# 查看你的IP
ipconfig  # Windows
ifconfig  # Mac/Linux

# 告诉同事访问
http://你的IP:5000
```

### 方法2: 内网穿透（推荐）

```bash
# 安装ngrok
# 访问 https://ngrok.com 下载

# 启动穿透
ngrok http 5000

# 获得公网链接，发给任何人都能访问！
```

## ❓ 常见问题

**Q: 端口被占用？**
```bash
python start_server.py --port 8080
```

**Q: 如何停止服务？**
```bash
按 Ctrl+C
```

**Q: 如何后台运行？**
```bash
nohup python app.py > server.log 2>&1 &
```
